# SyMo (переписан на JavaScript + CSS)

Проект SyMo полностью переписан с Python на **JavaScript** и использует CSS для оформления интерфейса.

## Технологии

- Node.js 20+
- JavaScript (ESM)
- CSS
- `systeminformation`
- `axios`

## Запуск

```bash
npm install
npm run start
```

## Проверка синтаксиса

```bash
npm run lint
```
